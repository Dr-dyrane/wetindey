# Design System & UX Standards

This directory specifies visual languages, layout containment rules, typography, icons, motion principles, and accessibility expectations for the WetinDey platform.

## Key Files

* **[APPLE-HIG-MAPPING.md](APPLE-HIG-MAPPING.md)**: **Apple HIG Adaptations** detailing WetinDey visual styling (borderless surfaces relying on contrasting background colors, smooth squircles `rounded-[24px]` for cards, and micro-interactions).
* **[ACCESSIBILITY.md](ACCESSIBILITY.md)**: Accessibility requirements for touch target sizes (minimum 44x44px, 48px preferred), screen readers, aria-attributes, and semantic tags.
* **[ICONOGRAPHY.md](ICONOGRAPHY.md)**: Permitted semantic iconography rules to prevent visual noise.
* **[MOTION-SYSTEM.md](MOTION-SYSTEM.md)**: Transition and slide-in motion guidelines for drawers and sheets.
* **[MOTION-IMPLEMENTATION-BASELINE.md](MOTION-IMPLEMENTATION-BASELINE.md)**: Implementation benchmarks for framer-motion/transitions.
